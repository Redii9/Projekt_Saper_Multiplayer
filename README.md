# Projekt gry Saper Multiplayer
## O projekcie
Jest to klasyczna gra saper z dodaniem do niej pojedynków multiplayer między dwoma graczami.
Wygrywa ten kto pierwszy odkryje bezpiecznie pola.

## Skład zespołu
- Adrian Rewera
- Kamil Raczyński

## Technologie
- Język programowania: Python wesja 3+
- Interfejs graficzny: Tkinter

## Architektura sieciowa
- Protokoły komunikacyjne: Sockets
- Architektura połączenia: Klient-serwer
- Model połączenia: LAN
